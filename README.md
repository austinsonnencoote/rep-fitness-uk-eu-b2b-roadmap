# REP Fitness UK/EU — Shopify B2B Implementation Roadmap

**Live site → https://austinsonnencoote.github.io/rep-fitness-uk-eu-b2b-roadmap/**

A microsite covering REP Fitness UK/EU's path to a B2B portal go-live on Shopify, including:

- **Use Case 01 — UK B2B Portal & Commercial Sign-Up:** how to build the portal, the sign-up form, and the Shopify Flow → HubSpot routing.
- **Launch Roadmap:** four-week plan from kick-off through UAT to **Go-Live on June 15**, with twice-weekly Shopify check-ins.

Designed to scale: V1 launches in the UK in GBP, then DE / FR / IT / ES are additive — same portal, new markets.
